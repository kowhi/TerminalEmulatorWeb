一个基于部标jtt808 2013版协议的终端模拟器

在部标系统开发过程中虽然可以使用设备进行一些调试，但是在很多业务场景下使用设备就显得捉襟见肘，比如： 围栏测试 压力测试 业务调试 其他...

为解决以上问题该项目应运而生
技能： Angular TypeScript Rx 其他 优点： 响应式 模块化 依赖注入 其他
技能： Net5.0 Socket EfCore Pgsql 其他 优点： 跨平台 语法糖 Lambda Asyn 其他

使用前请按照如下步骤操作

终端管理 - 添加车辆数据 线路管理 - 添加线路数据 任务管理 - 添加任务数据 任务列表 - 开始任务

注意：添加任务时请确保模拟器的服务端能够连接上任务中的服务端ip及端口
